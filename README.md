# Install
1. Copy and paste the `.env.exemple` file and rename it to `.env` 
2. Enter the information corresponding to you
   
    [Database Engine Support](https://github.com/sequelize/sequelize/blob/master/ENGINE.md)
3. Install npm
   ```cmd
      npm i
   ```
4. Run server
   ```
      npm start
   ```
